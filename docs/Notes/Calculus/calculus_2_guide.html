Power Rule Application

Find the derivative of \(f(x) = 3x^4 - 2x^3 + 5x - 7\).

Approach 1: Power Rule

Power Rule

If \(f(x) = x^n\), then \(f'(x) = nx^{n-1}\)
1
Apply Power Rule to Each Term

For \(3x^4\): multiply by exponent and reduce power by 1

$$\frac{d}{dx}(3x^4) = 3 \cdot 4x^{4-1} = 12x^3$$
2
Continue for All Terms
$$\frac{d}{dx}(-2x^3) = -2 \cdot 3x^{3-1} = -6x^2$$ $$\frac{d}{dx}(5x) = 5 \cdot 1x^{1-1} = 5$$ $$\frac{d}{dx}(-7) = 0$$
3
Combine Results
$$f'(x) = 12x^3 - 6x^2 + 5$$
The derivative of a constant is always zero!

Approach 2: First Principles

Definition of Derivative

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$
1
Set Up the Limit
$$f'(x) = \lim_{h \to 0} \frac{[3(x+h)^4 - 2(x+h)^3 + 5(x+h) - 7] - [3x^4 - 2x^3 + 5x - 7]}{h}$$
2
Expand and Simplify

After expanding and canceling, the limit yields the same result.

$$f'(x) = 12x^3 - 6x^2 + 5$$
This method is more tedious but proves the power rule works!

Approach 3: Term by Term Analysis

1
Identify Each Term Type
  • \(3x^4\) - polynomial term with coefficient
  • \(-2x^3\) - polynomial term with negative coefficient
  • \(5x\) - linear term
  • \(-7\) - constant term
2
Apply Appropriate Rules
$$\boxed{f'(x) = 12x^3 - 6x^2 + 5}$$
2

Product Rule

Find the derivative of \(g(x) = (2x^2 + 3)(x^3 - 4x)\).

Approach 1: Product Rule

Product Rule

If \(h(x) = f(x) \cdot g(x)\), then: $$h'(x) = f'(x) \cdot g(x) + f(x) \cdot g'(x)$$
1
Identify Components

Let \(f(x) = 2x^2 + 3\) and \(g(x) = x^3 - 4x\)

2
Find Individual Derivatives
$$f'(x) = 4x$$ $$g'(x) = 3x^2 - 4$$
3
Apply Product Rule
$$g'(x) = (4x)(x^3 - 4x) + (2x^2 + 3)(3x^2 - 4)$$ $$= 4x^4 - 16x^2 + 6x^4 - 8x^2 + 9x^2 - 12$$ $$= \boxed{10x^4 - 15x^2 - 12}$$

Approach 2: Expand First

1
Multiply Out
$$(2x^2 + 3)(x^3 - 4x) = 2x^5 - 8x^3 + 3x^3 - 12x$$ $$= 2x^5 - 5x^3 - 12x$$
2
Apply Power Rule
$$g'(x) = 10x^4 - 15x^2 - 12$$
Both methods give the same answer! Use product rule when expansion is messy.
3

Chain Rule

Find the derivative of \(h(x) = (3x^2 + 2x - 1)^5\).

Approach 1: Chain Rule

Chain Rule

If \(h(x) = f(g(x))\), then: $$h'(x) = f'(g(x)) \cdot g'(x)$$
1
Identify Outer and Inner Functions

Outer: \(f(u) = u^5\)

Inner: \(u = g(x) = 3x^2 + 2x - 1\)

2
Find Derivatives
$$f'(u) = 5u^4$$ $$g'(x) = 6x + 2$$
3
Apply Chain Rule
$$h'(x) = 5(3x^2 + 2x - 1)^4 \cdot (6x + 2)$$ $$= \boxed{5(6x + 2)(3x^2 + 2x - 1)^4}$$

Approach 2: Step-by-Step Breakdown

1
Take Derivative of Outer Function

Treat the inside as a single variable:

$$\frac{d}{dx}[(3x^2 + 2x - 1)^5] = 5(3x^2 + 2x - 1)^4 \cdot \frac{d}{dx}[3x^2 + 2x - 1]$$
2
Multiply by Inner Derivative
$$= 5(3x^2 + 2x - 1)^4 \cdot (6x + 2)$$
Remember: derivative of outside times derivative of inside!
Summary: Derivatives measure instantaneous rates of change. Master the power, product, quotient, and chain rules to differentiate any combination of functions efficiently.
