Section 2.3

Linear First-Order Equations ‚Äì Solution Strategies
Linear First-Order Differential Equations have the standard form \(\frac{dy}{dx} + P(x)y = f(x)\). The solution method uses an integrating factor \(\mu(x) = e^{\int P(x)dx}\) which transforms the equation into an exact differential. This technique is one of the most important in differential equations!
1

Basic Linear Equation

Find the general solution: $$5\frac{dy}{dx} + 30y = 6$$ Also find the largest interval I and identify any transient terms.

Approach 1: Standard Form Method

1
Write in Standard Form
$$5\frac{dy}{dx} + 30y = 6$$ $$\frac{dy}{dx} + 6y = \frac{6}{5}$$

So \(P(x) = 6\) and \(f(x) = \frac{6}{5}\)

2
Find Integrating Factor
$$\mu(x) = e^{\int 6 dx} = e^{6x}$$
3
Multiply and Integrate
$$e^{6x}\frac{dy}{dx} + 6e^{6x}y = \frac{6}{5}e^{6x}$$ $$\frac{d}{dx}[e^{6x}y] = \frac{6}{5}e^{6x}$$ $$e^{6x}y = \int \frac{6}{5}e^{6x}dx = \frac{6}{5} \cdot \frac{e^{6x}}{6} + C = \frac{e^{6x}}{5} + C$$
4
Solve for y
$$y = e^{-6x}\left(\frac{e^{6x}}{5} + C\right)$$ $$\boxed{y(x) = \frac{1}{5} + Ce^{-6x}}$$
5
Interval and Transient Terms

Interval: \((-\infty, \infty)\) (no singular points)

Transient term: \(Ce^{-6x}\) (decays to 0 as \(x \to \infty\))

Approach 2: Direct Integration

Integrating Factor Method

For \(\frac{dy}{dx} + P(x)y = f(x)\), multiply by \(\mu = e^{\int P dx}\)
1
Standard Form
$$\frac{dy}{dx} + 6y = \frac{6}{5}$$
2
Apply \(\mu = e^{6x}\)
$$\frac{d}{dx}[e^{6x}y] = \frac{6}{5}e^{6x}$$ $$e^{6x}y = \frac{1}{5}e^{6x} + C$$ $$\boxed{y = \frac{1}{5} + Ce^{-6x}}$$

Approach 3: Complete Analysis

1
Solution
$$y(x) = \frac{1}{5} + Ce^{-6x}$$
2
Verify

Differentiate:

$$\frac{dy}{dx} = -6Ce^{-6x}$$

Check:

$$5(-6Ce^{-6x}) + 30\left(\frac{1}{5} + Ce^{-6x}\right)$$ $$= -30Ce^{-6x} + 6 + 30Ce^{-6x} = 6$$ ‚úì
3
Analysis

Interval: \((-\infty, \infty)\)

Transient: \(Ce^{-6x}\) ‚Üí 0 as \(x \to \infty\)

Steady state: \(y = \frac{1}{5}\)

2

Polynomial Right Side

Find the general solution: $$y' = 2y + x^2 + 7$$

Approach 1: Integrating Factor Method

1
Standard Form
$$y' - 2y = x^2 + 7$$

So \(P(x) = -2\) and \(f(x) = x^2 + 7\)

2
Find \(\mu(x)\)
$$\mu(x) = e^{\int -2 dx} = e^{-2x}$$
3
Multiply and Integrate
$$\frac{d}{dx}[e^{-2x}y] = e^{-2x}(x^2 + 7)$$ $$e^{-2x}y = \int e^{-2x}(x^2 + 7)dx$$
4
Use Integration by Parts

For \(\int e^{-2x}x^2 dx\), use tabular integration:

$$\int e^{-2x}x^2 dx = -\frac{1}{2}e^{-2x}x^2 - \frac{1}{2}e^{-2x}x - \frac{1}{4}e^{-2x}$$

And:

$$\int 7e^{-2x}dx = -\frac{7}{2}e^{-2x}$$
5
Combine and Solve
$$e^{-2x}y = -\frac{1}{2}e^{-2x}x^2 - \frac{1}{2}e^{-2x}x - \frac{1}{4}e^{-2x} - \frac{7}{2}e^{-2x} + C$$ $$y = -\frac{1}{2}x^2 - \frac{1}{2}x - \frac{15}{4} + Ce^{2x}$$ $$\boxed{y(x) = Ce^{2x} - \frac{1}{2}x^2 - \frac{1}{2}x - \frac{15}{4}}$$

Approach 2: Detailed Steps

1
Setup
$$y' - 2y = x^2 + 7$$ $$\mu = e^{-2x}$$
2
Integrate \(\int x^2 e^{-2x}dx\)

Let \(u = x^2, dv = e^{-2x}dx\)

$$= -\frac{1}{2}x^2e^{-2x} + \int xe^{-2x}dx$$

Again: \(u = x, dv = e^{-2x}dx\)

$$= -\frac{1}{2}x^2e^{-2x} - \frac{1}{2}xe^{-2x} - \frac{1}{4}e^{-2x}$$
3
Final Answer
$$\boxed{y = Ce^{2x} - \frac{1}{2}x^2 - \frac{1}{2}x - \frac{15}{4}}$$

Transient: None (grows exponentially)

Interval: \((-\infty, \infty)\)

Approach 3: Verification

1
Differentiate Solution

If \(y = Ce^{2x} - \frac{1}{2}x^2 - \frac{1}{2}x - \frac{15}{4}\):

$$y' = 2Ce^{2x} - x - \frac{1}{2}$$
2
Check Original Equation
$$y' - 2y = 2Ce^{2x} - x - \frac{1}{2} - 2\left(Ce^{2x} - \frac{1}{2}x^2 - \frac{1}{2}x - \frac{15}{4}\right)$$ $$= 2Ce^{2x} - x - \frac{1}{2} - 2Ce^{2x} + x^2 + x + \frac{15}{2}$$ $$= x^2 + \frac{14}{2} = x^2 + 7$$ ‚úì
3

Finding P(x) and Integrating Factor

For: $$x\frac{dy}{dx} - y = x^2\sin(x)$$ Find P(x), the integrating factor, and the general solution.

Approach 1: Standard Procedure

1
Write in Standard Form
$$x\frac{dy}{dx} - y = x^2\sin(x)$$ $$\frac{dy}{dx} - \frac{1}{x}y = x\sin(x)$$

Therefore: \(\boxed{P(x) = -\frac{1}{x}}\)

2
Find Integrating Factor
$$\mu(x) = e^{\int P(x)dx} = e^{\int -\frac{1}{x}dx} = e^{-\ln|x|}$$ $$\boxed{\mu(x) = e^{\ln|x|^{-1}} = \frac{1}{x}}$$
3
Apply Integrating Factor
$$\frac{1}{x}\frac{dy}{dx} - \frac{1}{x^2}y = \sin(x)$$ $$\frac{d}{dx}\left[\frac{y}{x}\right] = \sin(x)$$ $$\frac{y}{x} = \int \sin(x)dx = -\cos(x) + C$$
4
Solve for y
$$\boxed{y(x) = -x\cos(x) + Cx}$$

Interval: \((0, \infty)\) or \((-\infty, 0)\) (singular at x = 0)

Transient: None

Approach 2: Detailed Analysis

Singular Points

When P(x) has discontinuities (like \(-\frac{1}{x}\) at x=0), the solution interval excludes those points.
1
Identify P(x)
$$\frac{dy}{dx} + P(x)y = f(x)$$ $$P(x) = -\frac{1}{x}, \quad f(x) = x\sin(x)$$
2
Integrating Factor
$$\int -\frac{1}{x}dx = -\ln|x|$$ $$\mu = e^{-\ln|x|} = |x|^{-1} = \frac{1}{|x|}$$

For simplicity, use \(\mu = \frac{1}{x}\) on appropriate intervals.

3
Solution
$$y = -x\cos(x) + Cx$$

Approach 3: Complete Solution

1
All Steps Together
$$P(x) = -\frac{1}{x}$$ $$\mu(x) = \frac{1}{x}$$ $$y(x) = -x\cos(x) + Cx$$
2
Verify

Differentiate:

$$y' = -\cos(x) + x\sin(x) + C$$

Check:

$$x(-\cos x + x\sin x + C) - (-x\cos x + Cx)$$ $$= -x\cos x + x^2\sin x + Cx + x\cos x - Cx$$ $$= x^2\sin x$$ ‚úì
3
Final Answers

Largest interval: \((0, \infty)\) or \((-\infty, 0)\)

Transient terms: NONE

Key Formula: For \(\frac{dy}{dx} + P(x)y = f(x)\), the solution is \(y = e^{-\int P dx}\left[\int f(x)e^{\int P dx}dx + C\right]\). Always check for singular points where P(x) is undefined‚Äîthese determine your interval of validity. Practice identifying transient terms (those that decay to zero as x ‚Üí ‚àû)!
