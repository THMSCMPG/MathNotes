Section 3.3

Systems of First-Order Differential Equations
This section introduces systems of differential equations where multiple quantities change simultaneously and interact. Applications include predator-prey models, chemical reactions, and interconnected tanks.
1

Coupled Tanks Problem

Two tanks each hold 100 gallons. Tank A initially has 20 lbs salt, Tank B has 0 lbs. Solution flows between them at 5 gal/min. Find salt in each tank over time.

Approach 1: System Setup

1
Define Variables

Let A(t) = salt in tank A, B(t) = salt in tank B

2
Rate Equations

For each tank, (rate in) - (rate out)

$$\frac{dA}{dt} = \frac{5B}{100} - \frac{5A}{100} = \frac{B-A}{20}$$ $$\frac{dB}{dt} = \frac{5A}{100} - \frac{5B}{100} = \frac{A-B}{20}$$
3
Notice Symmetry

Add equations

$$\frac{d(A+B)}{dt} = 0 \implies A + B = 20 \text{ (constant)}$$
4
Solve Reduced System

Substitute B = 20 - A

$$\frac{dA}{dt} = \frac{(20-A)-A}{20} = 1 - \frac{A}{10}$$
5
Find Solution

Solve first-order linear DE

$$A(t) = 10 + 10e^{-t/10}$$ $$B(t) = 10 - 10e^{-t/10}$$

Approach 2: Matrix Method Preview

1
Write in Matrix Form

System as matrix equation

$$\begin{pmatrix} A' \\ B' \end{pmatrix} = \begin{pmatrix} -1/20 & 1/20 \\ 1/20 & -1/20 \end{pmatrix}\begin{pmatrix} A \\ B \end{pmatrix}$$
2
Find Eigenvalues

Solve characteristic equation

$$\lambda_1 = 0, \quad \lambda_2 = -1/10$$
3
General Solution

Combine eigenvector solutions

Predator-Prey (Lotka-Volterra)

Rabbits (R) and foxes (F) satisfy: $$\frac{dR}{dt} = 2R - 0.5RF, \quad \frac{dF}{dt} = -F + 0.1RF$$

Approach 1: Equilibrium Analysis

1
Find Equilibria

Set both derivatives to zero

$$R(2 - 0.5F) = 0, \quad F(-1 + 0.1R) = 0$$ $$(0,0) \text{ and } (10, 4)$$
2
Interpret Equilibria

(0,0): extinction\n(10,4): coexistence

3
Analyze Stability

Linearize around (10,4)

4
Predict Behavior

Closed orbits (periodic solutions)

Approach 2: Phase Plane Analysis

1
Find Nullclines

Where dR/dt = 0 and dF/dt = 0

$$R = 0 \text{ or } F = 4 \text{ (R-nullcline)}$$ $$F = 0 \text{ or } R = 10 \text{ (F-nullcline)}$$
2
Sketch Direction Field

Determine flow in each region

3
Identify Cycles

Solutions orbit around (10,4)

4
Biological Meaning

Population oscillations are natural

Chemical Reactions

Chemical reaction A + B → C has rate k[A][B]. If A(0) = a, B(0) = b, find concentrations over time.

Approach 1: System from Reaction

1
Write Rate Laws

Based on stoichiometry

$$\frac{dA}{dt} = -k[A][B]$$ $$\frac{dB}{dt} = -k[A][B]$$ $$\frac{dC}{dt} = k[A][B]$$
2
Find Conservation Law

A + C = a (constant), B + C = b

3
Reduce to Single Equation

Substitute to eliminate variables

$$\frac{dA}{dt} = -k·A·(b-a+A)$$
4
Solve (if a ≠ b)

Use partial fractions
