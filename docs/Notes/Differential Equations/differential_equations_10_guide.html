Section 4.3

Constant Coefficients and Characteristic Equations
This section covers the most important method for solving homogeneous linear DEs with constant coefficients: the characteristic equation method. We learn to handle all three cases: distinct real roots, repeated roots, and complex roots.
1

Distinct Real Roots

Solve y'' - 5y' + 6y = 0

Approach 1: Characteristic Equation Method

1
Assume Exponential Solution

Try y = e^(rx)

2
Form Characteristic Equation

Substitute into DE

$$r^2e^{rx} - 5re^{rx} + 6e^{rx} = 0$$ $$r^2 - 5r + 6 = 0$$
3
Solve for r

Factor or use quadratic formula

$$(r-2)(r-3) = 0 \implies r_1 = 2, r_2 = 3$$
4
Write General Solution

Linear combination of solutions

$$y = c_1e^{2x} + c_2e^{3x}$$
5
Verify Linear Independence

W(e^(2x), e^(3x)) = e^(5x) ≠ 0

Approach 2: Initial Value Problem

1
Start with General Solution

y = c₁e^(2x) + c₂e^(3x)

2
Apply y(0) = 1

c₁ + c₂ = 1

3
Compute y'

y' = 2c₁e^(2x) + 3c₂e^(3x)

4
Apply y'(0) = 0

2c₁ + 3c₂ = 0

5
Solve System

c₁ = 3, c₂ = -2

$$y = 3e^{2x} - 2e^{3x}$$
2

Repeated Roots

Solve y'' - 6y' + 9y = 0

Approach 1: Repeated Root Formula

1
Characteristic Equation

r² - 6r + 9 = 0

$$(r-3)^2 = 0 \implies r = 3 \text{ (multiplicity 2)}$$
2
First Solution

Standard exponential

$$y_1 = e^{3x}$$
3
Second Solution

Multiply by x for repeated root

$$y_2 = xe^{3x}$$
4
General Solution

Combine solutions

$$y = c_1e^{3x} + c_2xe^{3x} = e^{3x}(c_1 + c_2x)$$
5
Why This Works

Reduction of order would give same result

Approach 2: Higher Multiplicity

1
General Rule

For root r with multiplicity k

$$e^{rx}, xe^{rx}, x^2e^{rx}, \ldots, x^{k-1}e^{rx}$$
2
Example: Triple Root

If (r-2)³ = 0

$$y = e^{2x}(c_1 + c_2x + c_3x^2)$$
3
Verify Independence

These are always linearly independent

3

Complex Roots

Solve y'' + 4y' + 13y = 0

Approach 1: Euler's Formula Approach

1
Characteristic Equation

r² + 4r + 13 = 0

$$r = \frac{-4 \pm \sqrt{16-52}}{2} = \frac{-4 \pm 6i}{2} = -2 \pm 3i$$
2
Identify α and β

Complex roots: r = α ± βi

$$\alpha = -2, \quad \beta = 3$$
3
Real-Valued Solutions

Use Euler's formula e^(ix) = cos(x) + i sin(x)

$$y_1 = e^{\alpha x}\cos(\beta x) = e^{-2x}\cos(3x)$$ $$y_2 = e^{\alpha x}\sin(\beta x) = e^{-2x}\sin(3x)$$
4
General Solution

Linear combination

$$y = e^{-2x}(c_1\cos(3x) + c_2\sin(3x))$$

Approach 2: Complex Exponentials

1
Complex Solutions

y = e^((-2+3i)x) and y = e^((-2-3i)x)

2
Apply Euler's Formula

e^((α+iβ)x) = e^(αx)(cos(βx) + i sin(βx))

3
Take Real and Imaginary Parts

Both are real solutions

$$\text{Real part: } e^{-2x}\cos(3x)$$ $$\text{Imaginary part: } e^{-2x}\sin(3x)$$
4
Why This Works

Superposition principle for linear DEs

Approach 3: Oscillatory Behavior

1
Analyze Solution Structure

e^(αx)[c₁cos(βx) + c₂sin(βx)]

2
Envelope

e^(-2x) causes exponential decay

3
Oscillation

cos(3x) and sin(3x) cause oscillation

$$\text{Period} = \frac{2\pi}{\beta} = \frac{2\pi}{3}$$
4
Behavior

Damped oscillations approaching 0 as x → ∞

4

Higher-Order Equations

Solve y^(4) - 5y'' + 4y = 0

Approach 1: Fourth-Order Characteristic

1
Characteristic Equation

r⁴ - 5r² + 4 = 0

2
Factor as Quadratic in r²

Let u = r²

$$(u-1)(u-4) = 0 \implies u = 1, 4$$
3
Solve for r

r² = 1 or r² = 4

$$r = \pm 1, \pm 2$$
4
General Solution

Four distinct real roots

$$y = c_1e^x + c_2e^{-x} + c_3e^{2x} + c_4e^{-2x}$$
5
Alternative Form

Using hyperbolic functions

$$y = A\cosh(x) + B\sinh(x) + C\cosh(2x) + D\sinh(2x)$$
Summary: Each problem can be approached from multiple angles. Understanding different methods helps develop deeper mathematical intuition and provides flexibility for solving similar problems in the future. Practice switching between these perspectives to strengthen your problem-solving skills!
