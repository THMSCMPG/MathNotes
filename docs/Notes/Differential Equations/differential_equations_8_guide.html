1

Linear Independence and Wronskian

Determine if {e^x, e^(2x), e^(3x)} is linearly independent on (-∞, ∞)

Approach 1: Wronskian Method

1
Compute the Wronskian

For three functions, 3×3 determinant

$$W(e^x, e^{2x}, e^{3x}) = \begin{vmatrix} e^x & e^{2x} & e^{3x} \\ e^x & 2e^{2x} & 3e^{3x} \\ e^x & 4e^{2x} & 9e^{3x} \end{vmatrix}$$
2
Factor Out Common Terms

Factor e^(x+2x+3x) = e^(6x)

$$W = e^{6x}\begin{vmatrix} 1 & 1 & 1 \\ 1 & 2 & 3 \\ 1 & 4 & 9 \end{vmatrix}$$
3
Evaluate Determinant

Expand along first row

$$= e^{6x}[1(18-12) - 1(9-3) + 1(4-2)] = e^{6x}(6-6+2) = 2e^{6x}$$
4
Conclude

W ≠ 0 for all x, so functions are linearly independent

Approach 2: Definition Check

1
Set Up Linear Combination

Test if c₁e^x + c₂e^(2x) + c₃e^(3x) = 0 implies all c_i = 0

2
Evaluate at Special Points

Try x = 0, x = ln2, etc.

$$x=0: c_1 + c_2 + c_3 = 0$$
3
Create System

Get more equations by differentiation or evaluation

4
Solve System

Show only trivial solution exists

Fundamental Set of Solutions

Verify that {e^(-x), xe^(-x), x²e^(-x)} forms a fundamental set for y''' + 3y'' + 3y' + y = 0

Approach 1: Complete Verification

1
Verify Each is a Solution

Substitute each function into the DE

$$y_1 = e^{-x}: y_1''' + 3y_1'' + 3y_1' + y_1 = -e^{-x} + 3e^{-x} - 3e^{-x} + e^{-x} = 0 \checkmark$$
2
Check Linear Independence

Compute Wronskian at any point, say x = 0

$$W(0) = \begin{vmatrix} 1 & 0 & 0 \\ -1 & 1 & 0 \\ 1 & -2 & 2 \end{vmatrix} = 2 \neq 0$$
3
Count Functions

Have 3 functions for 3rd-order DE

4
Conclude

Forms a fundamental set

Approach 2: General Solution

1
Form General Solution

Linear combination of fundamental set

$$y = c_1e^{-x} + c_2xe^{-x} + c_3x^2e^{-x}$$
2
Factor Common Term

Simplify

$$y = e^{-x}(c_1 + c_2x + c_3x^2)$$
3
Apply Initial Conditions

If given IVP, solve for constants

Existence and Uniqueness

For y'' + p(x)y' + q(x)y = f(x) with y(0) = 1, y'(0) = 2, when is there a unique solution?

Approach 1: Theorem Application

1
State Theorem

Existence and Uniqueness Theorem

If p(x), q(x), and f(x) are continuous on interval I containing x₀, then IVP has unique solution on I
2
Check Conditions

Verify continuity of coefficients

3
Identify Interval

Find largest interval where all are continuous

4
Conclude

Unique solution exists on that interval

Superposition Principle

If y₁ and y₂ satisfy y'' + 4y = x², what can you say about c₁y₁ + c₂y₂?

Approach 1: Direct Verification

1
Form Linear Combination

Let y = c₁y₁ + c₂y₂

2
Compute Derivatives

y'' = c₁y₁'' + c₂y₂''

3
Substitute into DE

y'' + 4y = c₁(y₁'' + 4y₁) + c₂(y₂'' + 4y₂)

4
Use Given Information

Both parentheses equal x²

$$= c_1x^2 + c_2x^2 = (c_1+c_2)x^2$$
5
Conclusion

Linear combination satisfies DE only if c₁+c₂ = 1 (inhomogeneous!)

Approach 2: Homogeneous vs Inhomogeneous

1
Note This is NOT Homogeneous

Right side is x², not 0

2
Superposition for Homogeneous

Works without restriction on constants

3
Particular Solutions

For inhomogeneous, need y₁ - y₂ satisfies homogeneous
