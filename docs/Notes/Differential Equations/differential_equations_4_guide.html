Section 2.6

Euler Methods and Numerical Approximations
This section covers numerical approximation methods for solving differential equations when exact solutions are difficult or impossible to find. We focus on Euler's method and improved Euler method, learning how to implement these techniques and assess their accuracy.
1

Standard Euler Method

Use Euler's method with h = 0.1 to approximate y(1.5) for the IVP: $$y' = 2xy, \quad y(1) = 1$$

Approach 1: Step-by-Step Iteration

1
Understand Euler's Formula

The basic Euler iteration formula

$$y_{n+1} = y_n + h \cdot f(x_n, y_n)$$
2
Identify Given Information

Initial condition: (x₀, y₀) = (1, 1)\nStep size: h = 0.1\nFunction: f(x, y) = 2xy

3
First Iteration (n=0)

Calculate from x = 1.0 to x = 1.1

$$x_1 = 1.1$$ $$y_1 = y_0 + h \cdot f(x_0, y_0) = 1 + 0.1(2)(1)(1) = 1.2$$
4
Second Iteration (n=1)

Calculate from x = 1.1 to x = 1.2

$$x_2 = 1.2$$ $$y_2 = 1.2 + 0.1(2)(1.1)(1.2) = 1.464$$
5
Continue to Target

Repeat until x = 1.5 (5 iterations total)

Approach 2: Tabular Organization

1
Set Up Table

Create columns for: n, xₙ, yₙ, f(xₙ, yₙ), h·f(xₙ, yₙ)

2
Fill Initial Row

n = 0: x₀ = 1, y₀ = 1, f(1, 1) = 2, h·f = 0.2

3
Compute Each Row

Each new yₙ₊₁ = previous yₙ + h·f from previous row

4
Read Final Answer

The y-value when x = 1.5

Approach 3: Error Analysis

1
Find Exact Solution

This is a separable equation

$$\frac{dy}{y} = 2x\,dx \implies \ln|y| = x^2 + C$$ $$y(1) = 1 \implies C = -1$$ $$y = e^{x^2-1}$$
2
Calculate Exact Value

At x = 1.5

$$y(1.5) = e^{1.5^2-1} = e^{1.25} \approx 3.4903$$
3
Compare with Approximation

Euler gives approximately y(1.5) ≈ 2.7095

$$\text{Absolute Error} = |3.4903 - 2.7095| \approx 0.7808$$ $$\text{Relative Error} \approx 22.4\%$$
4
Observe Error Behavior

Smaller h reduces error (try h = 0.05 for better accuracy)

2

Improved Euler Method

Apply the improved Euler method (Heun's method) to the same problem and compare accuracy

Approach 1: Predictor-Corrector Approach

1
Predictor Step

Use standard Euler to predict

$$y^*_{n+1} = y_n + h \cdot f(x_n, y_n)$$
2
Corrector Step

Average the slopes at both endpoints

$$y_{n+1} = y_n + \frac{h}{2}[f(x_n, y_n) + f(x_{n+1}, y^*_{n+1})]$$
3
Example: First Iteration

From (1, 1) to (1.1, ?)

$$y^*_1 = 1 + 0.1(2)(1)(1) = 1.2$$ $$y_1 = 1 + \frac{0.1}{2}[2(1)(1) + 2(1.1)(1.2)]$$ $$y_1 = 1 + 0.05[2 + 2.64] = 1.232$$
4
Compare Accuracy

Improved Euler is more accurate for same h

Approach 2: Geometric Interpretation

1
Standard Euler

Uses slope at left endpoint only

2
Improved Euler

Averages slopes at left and right endpoints

3
Visual Understanding

This gives a better approximation of the curve

4
Result

Significantly reduces error with minimal extra computation

Approach 2: Error Estimation

1
Local Truncation Error

Error in single step

$$\text{LTE} = O(h^2) \text{ for Euler}$$ $$\text{LTE} = O(h^3) \text{ for Improved Euler}$$
2
Global Error

Accumulated error over interval

$$\text{GE} = O(h) \text{ for Euler}$$ $$\text{GE} = O(h^2) \text{ for Improved Euler}$$
3
Practical Implication

Need very small h for high accuracy with standard Euler
