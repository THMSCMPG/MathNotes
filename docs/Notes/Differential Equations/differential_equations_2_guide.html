Section 2.2

Separable Variables ‚Äì Solution Strategies
Separation of Variables is a fundamental technique for solving first-order differential equations. The method works when you can write the equation in the form \(g(y)dy = h(x)dx\), allowing you to integrate each side independently. This guide presents multiple solution approaches for each problem to help you develop flexibility and deeper understanding.
1

Basic Quadratic Separation

Solve the differential equation by separation of variables: $$dy - (y - 7)^2 dx = 0$$

Approach 1: Direct Separation

1
Separate Variables
$$dy = (y-7)^2 dx$$ $$\frac{dy}{(y-7)^2} = dx$$
2
Integrate Both Sides
$$\int (y-7)^{-2} dy = \int dx$$ $$\frac{(y-7)^{-1}}{-1} = x + C$$ $$-\frac{1}{y-7} = x + C$$
3
Solve for y
$$y - 7 = -\frac{1}{x+C}$$ $$\boxed{y = 7 - \frac{1}{x+C}}$$

Approach 2: Substitution Method

1
Make Substitution

Let \(u = y - 7\), then \(du = dy\)

$$du = u^2 dx$$ $$u^{-2} du = dx$$
2
Integrate
$$\int u^{-2} du = \int dx$$ $$-u^{-1} = x + C$$
3
Back-Substitute
$$-\frac{1}{y-7} = x + C$$ $$\boxed{y = 7 - \frac{1}{x+C}}$$

Approach 3: With Verification

1
Solve
$$\int \frac{dy}{(y-7)^2} = \int dx$$ $$-\frac{1}{y-7} = x + C$$ $$y = 7 - \frac{1}{x+C}$$
2
Verify

Differentiate: \(y = 7 - (x+C)^{-1}\)

$$\frac{dy}{dx} = 0 - (-1)(x+C)^{-2} = \frac{1}{(x+C)^2}$$

Check: \((y-7)^2 = \left(-\frac{1}{x+C}\right)^2 = \frac{1}{(x+C)^2}\) ‚úì

2

Exponential Equation

Solve: $$e^{xy} \frac{dy}{dx} = e^{-y} + e^{-6x - y}$$

Approach 1: Factor and Separate

1
Factor Right Side
$$e^{xy} \frac{dy}{dx} = e^{-y}(1 + e^{-6x})$$
2
Separate Variables
$$\frac{e^{xy}}{e^{-y}} dy = (1 + e^{-6x}) dx$$ $$e^{xy+y} dy = (1 + e^{-6x}) dx$$ $$e^{y(x+1)} dy = (1 + e^{-6x}) dx$$
The left side contains both x and y, making traditional separation impossible. This equation may require an implicit solution or numerical methods.

Approach 2: Using Exponential Properties

1
Rewrite
$$\frac{dy}{dx} = \frac{e^{-y} + e^{-6x-y}}{e^{xy}}$$ $$\frac{dy}{dx} = e^{-y-xy} + e^{-6x-y-xy}$$ $$\frac{dy}{dx} = e^{-y(1+x)}(1 + e^{-6x})$$
2
Attempt Separation
$$e^{y(1+x)} dy = (1 + e^{-6x}) dx$$
3
Integrate Right Side
$$\int (1 + e^{-6x}) dx = x - \frac{1}{6}e^{-6x} + C$$

Left side cannot be integrated in elementary form.

Approach 3: Recognition

Important Note

When \(e^{xy}\) appears, the equation is generally not separable unless there's a special cancellation.
This problem may have a typo or require implicit solution. Verify the problem statement or consult your textbook's answer key.
3

Rational Function

Solve: $$\frac{dy}{dx} = \frac{2y + 7}{8x + 9}$$

Approach 1: Logarithmic Integration

1
Separate
$$\frac{dy}{2y+7} = \frac{dx}{8x+9}$$
2
Integrate Using \(\int \frac{1}{ax+b}dx = \frac{1}{a}\ln|ax+b|\)
$$\frac{1}{2}\ln|2y+7| = \frac{1}{8}\ln|8x+9| + C$$
3
Solve for y
$$\ln|2y+7| = \frac{1}{4}\ln|8x+9| + K$$ $$|2y+7| = e^K |8x+9|^{1/4}$$ $$2y+7 = A(8x+9)^{1/4}$$ $$\boxed{y = \frac{A(8x+9)^{1/4} - 7}{2}}$$

Approach 2: Using Power Form

1
After Integration
$$\frac{1}{2}\ln|2y+7| = \frac{1}{8}\ln|8x+9| + C$$
2
Multiply by 2
$$\ln|2y+7| = \frac{1}{4}\ln|8x+9| + 2C$$ $$\ln|2y+7| = \ln|8x+9|^{1/4} + K$$
3
Exponentiate
$$|2y+7| = e^K (8x+9)^{1/4}$$ $$\boxed{y = \frac{A\sqrt[4]{8x+9} - 7}{2}}$$

Approach 3: Complete with Verification

1
Full Solution
$$\int \frac{dy}{2y+7} = \int \frac{dx}{8x+9}$$ $$\frac{1}{2}\ln|2y+7| = \frac{1}{8}\ln|8x+9| + C$$ $$y = \frac{A(8x+9)^{1/4} - 7}{2}$$
2
Verify

Differentiate \(y\):

$$\frac{dy}{dx} = \frac{A}{2} \cdot \frac{1}{4}(8x+9)^{-3/4} \cdot 8 = \frac{A}{(8x+9)^{3/4}}$$

Check RHS:

$$\frac{2y+7}{8x+9} = \frac{A(8x+9)^{1/4}}{8x+9} = \frac{A}{(8x+9)^{3/4}}$$

Match! ‚úì

4

Logistic Equation with Partial Fractions

$$\frac{dP}{dt} = P - P^2$$ Find: (a) \(\int \frac{dP}{g(P)}\) and \(\int h(t)dt\), then (b) solve the equation.

Approach 1: Partial Fractions

1
Identify Functions
$$g(P) = P - P^2 = P(1-P)$$ $$h(t) = 1$$
2
Partial Fractions
$$\frac{1}{P(1-P)} = \frac{A}{P} + \frac{B}{1-P}$$

Solving: \(A = 1, B = 1\)

$$\int \frac{dP}{P(1-P)} = \int \left(\frac{1}{P} + \frac{1}{1-P}\right) dP$$ $$= \ln|P| - \ln|1-P| + C = \boxed{\ln\left|\frac{P}{1-P}\right| + C}$$
3
Other Integral
$$\int h(t)dt = \boxed{t + C}$$
4
Solve
$$\ln\left|\frac{P}{1-P}\right| = t + C$$ $$\frac{P}{1-P} = Ae^t$$ $$\boxed{P = \frac{Ae^t}{1 + Ae^t}}$$

Approach 2: Logistic Form

Logistic Equation

\(\frac{dP}{dt} = P(1-P)\) is the standard logistic growth equation.
1
Standard Form

This is logistic growth with \(r = 1\) and \(K = 1\).

General solution:

$$P(t) = \frac{KP_0 e^{rt}}{K + P_0(e^{rt} - 1)} = \frac{P_0 e^t}{1 + P_0(e^t - 1)}$$
2
Simplify
$$\boxed{P = \frac{Ce^t}{1 + Ce^t}}$$

Approach 3: Complete Derivation

1
Separate and Integrate
$$\frac{dP}{P(1-P)} = dt$$ $$\int \left(\frac{1}{P} + \frac{1}{1-P}\right)dP = \int dt$$ $$\ln|P| - \ln|1-P| = t + C$$
2
Exponentiate
$$\ln\left|\frac{P}{1-P}\right| = t + C$$ $$\frac{P}{1-P} = Ke^t$$
3
Solve for P
$$P = Ke^t(1-P)$$ $$P = Ke^t - PKe^t$$ $$P(1 + Ke^t) = Ke^t$$ $$\boxed{P = \frac{Ke^t}{1 + Ke^t} = \frac{1}{1 + K^{-1}e^{-t}}}$$
Master Strategy: For separation of variables: (1) Rearrange to get all y-terms with dy and all x-terms with dx, (2) Integrate both sides, (3) Solve for y explicitly if possible, (4) Always verify by differentiation. Practice recognizing when to use partial fractions, substitution, or logarithmic integration!
