Section 3.2

Logistic Growth and Population Dynamics
This section studies nonlinear models, particularly the logistic equation for population growth with carrying capacity, and other biological and physical applications of nonlinear differential equations.
1

Logistic Growth Model

Solve the logistic equation $$\frac{dP}{dt} = 0.05P\left(1 - \frac{P}{1000}\right)$$ with P(0) = 100

Approach 1: Separation of Variables

1
Separate Variables

Rearrange the equation

$$\frac{dP}{P(1 - P/K)} = r\,dt$$
2
Partial Fractions

Decompose the left side where K = 1000, r = 0.05

$$\frac{1}{P(1-P/K)} = \frac{1}{P} + \frac{1/K}{1-P/K}$$
3
Integrate Both Sides

Integrate

$$\ln|P| - \ln|K-P| = rt + C$$
4
Solve for P

Apply initial condition P(0) = 100

$$P(t) = \frac{1000}{1 + 9e^{-0.05t}}$$
5
Analyze Behavior

As t → ∞, P → 1000 (carrying capacity)

Approach 2: Phase Line Analysis

1
Find Equilibria

Set dP/dt = 0

$$P = 0 \text{ (unstable)}, \quad P = 1000 \text{ (stable)}$$
2
Determine Stability

Test points between equilibria

3
Sketch Phase Portrait

All solutions with P(0) > 0 approach 1000

4
Interpret

Population always tends to carrying capacity

Approach 3: Logistic Formula Direct Application

1
Standard Logistic Solution

For general logistic equation

$$P(t) = \frac{K}{1 + Ae^{-rt}}$$
2
Find A

Use initial condition

$$A = \frac{K - P_0}{P_0} = \frac{1000-100}{100} = 9$$
3
Write Final Solution

Substitute all values

$$P(t) = \frac{1000}{1 + 9e^{-0.05t}}$$
2

Harvesting Model

A population grows logistically with r = 0.04 and K = 500. If h fish per year are harvested, find critical harvest rate.

Approach 1: Modified Logistic Equation

1
Set Up Model

Add harvesting term

$$\frac{dP}{dt} = rP\left(1-\frac{P}{K}\right) - h$$
2
Find Equilibria

Set dP/dt = 0

$$0.04P(1-P/500) - h = 0$$
3
Critical Harvest

Maximum sustainable harvest

$$h_{max} = \frac{rK}{4} = \frac{0.04 \times 500}{4} = 5 \text{ fish/year}$$
4
Interpret

If h > 5, population goes extinct

3

Competition Model

Two species compete: $$\frac{dx}{dt} = x(2-x-y), \quad \frac{dy}{dt} = y(1-x-0.5y)$$

Approach 1: Find Equilibrium Points

1
Set Both Derivatives to Zero

Solve system

$$x(2-x-y) = 0, \quad y(1-x-0.5y) = 0$$
2
Find All Equilibria

Four equilibrium points

$$(0,0), (2,0), (0,2), (1.33, 0.67)$$
3
Analyze Stability

Use Jacobian or phase plane analysis

4
Interpret

Coexistence or competitive exclusion?
