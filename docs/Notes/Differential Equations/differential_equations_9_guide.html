Section 4.2

Finding a Second Solution
This section presents the reduction of order method for finding a second linearly independent solution to a second-order linear homogeneous DE when one solution is known. This technique is essential when the characteristic equation approach doesn't work or we need to build solutions systematically.
1

Basic Reduction of Order

Given that y₁ = x² is a solution to x²y'' - 3xy' + 4y = 0, find a second linearly independent solution.

Approach 1: Reduction of Order Formula

1
Set Up y₂

Assume y₂ = v(x)y₁ = v(x)x²

2
Compute Derivatives

y₂' = v'x² + 2vx\ny₂'' = v''x² + 4v'x + 2v

3
Substitute into DE

x²(v''x² + 4v'x + 2v) - 3x(v'x² + 2vx) + 4vx² = 0

4
Simplify

Collect terms (the v terms cancel!)

$$x^4v'' + x^3v' = 0$$
5
Substitute w = v'

First-order DE for w

$$x^4w' + x^3w = 0 \implies w' = -\frac{w}{x}$$
6
Solve for w

Separable equation

$$w = \frac{C}{x}$$
7
Integrate for v

v = ∫(C/x)dx = C ln|x| + D

8
Form y₂

Take C = 1, D = 0

$$y_2 = x^2\ln|x|$$

Approach 2: Direct Formula Application

1
Standard Formula

For ay'' + by' + cy = 0 with known y₁

$$y_2 = y_1 \int \frac{e^{-\int P\,dx}}{y_1^2}\,dx$$ where P = b/a
2
Identify Coefficients

Rewrite: y'' - (3/x)y' + (4/x²)y = 0

$$P(x) = -\frac{3}{x}, \quad y_1 = x^2$$
3
Compute Integral

e^(-∫P dx) = e^(3ln|x|) = x³

$$y_2 = x^2\int\frac{x^3}{x^4}dx = x^2\int\frac{1}{x}dx = x^2\ln|x|$$
2

Wronskian Method Connection

Use the Wronskian to find y₂ given y₁ = sin(x) for y'' + y = 0

Approach 1: Abel's Theorem

1
Abel's Formula

For y'' + p(x)y' + q(x)y = 0

$$W(x) = Ce^{-\int p(x)\,dx}$$
2
For This Problem

p(x) = 0, so W(x) = C (constant)

3
Set Up Wronskian

W(y₁, y₂) = y₁y₂' - y₁'y₂

$$W = \sin(x)y_2' - \cos(x)y_2 = C$$
4
Solve for y₂'

First-order linear DE

$$y_2' - \cot(x)y_2 = C\csc(x)$$
5
Integrate

Choose C = 1

$$y_2 = \cos(x)$$
6
Verify

Check linear independence: W(sin x, cos x) = 1 ≠ 0

3

Higher-Order Reduction

Given y₁ = e^x for y''' - 3y'' + 3y' - y = 0, find the general solution.

Approach 1: Successive Reduction

1
First Reduction

Find y₂ = ve^x using reduction of order

$$y_2 = xe^x$$
2
Check Linear Independence

W(e^x, xe^x) ≠ 0

3
Need Third Solution

Use y₃ = wy₁ or y₃ = wy₂

4
Alternative: Factor

Notice characteristic equation (r-1)³ = 0

$$y = e^x(c_1 + c_2x + c_3x^2)$$
Summary: Each problem can be approached from multiple angles. Understanding different methods helps develop deeper mathematical intuition and provides flexibility for solving similar problems in the future. Practice switching between these perspectives to strengthen your problem-solving skills!
