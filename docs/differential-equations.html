Matrix Form

where x is vector of unknowns, A is coefficient matrix

Eigenvalue Method (Homogeneous)

For x' = Ax:

  1. Find eigenvalues Î» and eigenvectors v of A
  2. Solutions: x(t) = e^(Î»t)v
  3. General solution is linear combination

Complex eigenvalues: Use Euler's formula to get real solutions

Phase Plane Analysis

For 2D system x' = f(x,y), y' = g(x,y):

  • Equilibrium: Points where f = g = 0
  • Trajectories: Solution curves in xy-plane
  • Stability: Analyze eigenvalues at equilibria

Stability Classification

  • Both eigenvalues negative: stable node
  • Both eigenvalues positive: unstable node
  • Opposite signs: saddle point (unstable)
  • Complex with negative real part: stable spiral
  • Pure imaginary: center (neutrally stable)
