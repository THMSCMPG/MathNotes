Mean Value Theorem

If f is continuous on [a,b] and differentiable on (a,b), then there exists c âˆˆ (a,b) where:

f'(c) = [f(b) - f(a)]/(b - a)

Interpretation: At some point, the instantaneous rate equals the average rate over the interval.

Critical Points

x = c is a critical point if f'(c) = 0 or f'(c) does not exist.

First Derivative Test:

  • f' changes from + to - at c: local max
  • f' changes from - to + at c: local min
  • f' doesn't change sign: inflection point

Second Derivative Test

If f'(c) = 0:

  • f''(c) > 0: local minimum
  • f''(c) < 0: local maximum
  • f''(c) = 0: test inconclusive

Concavity: f'' > 0 means concave up, f'' < 0 means concave down

Optimization

Strategy:

  1. Identify quantity to optimize (write as function)
  2. Find domain constraints
  3. Locate critical points: f'(x) = 0
  4. Evaluate at critical points and endpoints
  5. Verify with second derivative or first derivative test

Rolle's Theorem

If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then there exists c âˆˆ (a,b) where f'(c) = 0.

Significance: Foundation for proving the Mean Value Theorem

Extreme Value Theorem

If f is continuous on closed interval [a,b], then f attains both absolute maximum and absolute minimum values on [a,b].

Finding absolute extrema:

  1. Find critical points in (a,b)
  2. Evaluate f at critical points and endpoints
  3. Largest value is absolute max, smallest is absolute min
